  • マイページにて採択通知書確認
  • 見積書の添付は採択者全員必須
  • 修正依頼がある場合は申請内容を商工会コメントに基づき修正

差戻し対応の流れ

  • 商工会から差戻しがあれば「商工会からのコメント」を確認

  • 商工会の承認後に補助金事務局へ再提出される

申請システム操作

  • マイページから「公募・交付申請を開始する」をクリック

  • 資金調達方法の金額と必要事項を入力
  • 修正完了後、「修正したことを商工会へ通知する」ボタンで再提出
