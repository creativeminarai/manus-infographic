第19回公募要領第5版

「地域の小規模事業者の持続的な成長を支援する販路開拓および生産性向上補助金、最大200万円まで支援可能。」

資料の要旨

第19回小規模事業者持続化補助金（一般型 通常枠）は、物価高騰や賃上げ、インボイス制度などの制度変化に対応するため、小規模事業者及び一定要件を満たす特定非営利活動法人を対象に、自社で策定した経営計画に基づく販路開拓や業務効率化の取組に対し、経費の一部を補助し地域の雇用創出と産業の持続的発展を図る事業支援制度です。補助金額は基本上限50万円で、インボイス特例や賃金引上げ特例適用により最大200万円まで上乗せが可能です。申請は電子システムのみで受付け、商工会・商工会議所の支援を必須とするほか、不正受給防止のため厳格な申請要件が設けられています。申請締切は2026年4月30日、補助対象期間は交付決定日から2027年6月30日までです。

注目すべき重要点

  • 自ら経営計画を策定し販路開拓や生産性向上の取組を実施する小規模事業者が対象
  • 基本補助上限は50万円、インボイスや賃上げ特例により最大200万円まで増額可能
  • 審査により不採択・交付取消のリスクがあり、虚偽申請や高額アドバイス料金に注意
  • 申請は電子申請システムで、商工会・商工会議所の支援・事業支援計画書が必須
  • 補助金は後払い、自己負担が必要で、途中の事業内容変更等は減額・取消の対象

申請対象・主な要件

  • 日本国内に所在する小規模事業者または一定要件を満たす特定非営利活動法人
  • 業種別従業員数基準を満たすこと（例：製造業20人以下、商業・サービス業5人以下等）
  • 資本金5億円以上の法人が100％保有する企業は対象外
  • 過去の補助金実績報告義務を完了していること
  • インボイス発行事業者登録、賃上げ特例適用は申請時に選択・証明書類提出が必要
  • 補助率は2/3、赤字賃上げ事業者は3/4（賃上げ特例適用時）

詳細分析・重要項目解説

補助対象経費

補助対象経費は、策定した経営計画に基づき補助事業を遂行するために「支払完了かつ証憑確認可能な経費」で、下記8区分が対象です：
- 機械装置等費（事業遂行に必要で新品または条件を満たす中古品購入）
- 広報費
- ウェブサイト関連費
- 展示会等出展費（オンライン含む）
- 旅費
- 新商品開発費
- 借料
- 委託・外注費

審査のポイント

  • 経営計画に基づく実効性のある販路開拓等の取組か
  • 商工会議所・商工会の支援を確実に受けているか
  • 虚偽申告の有無（特に第三者支援の有無と金額の正確な報告）
  • 過去の補助事業報告義務の履行有無

事業計画書の書き方

  • 持続的な成長を見据えた明確な販路開拓・生産性向上施策を明示
  • 第三者支援の有無及び具体内容を正確に記載
  • インボイス特例、賃上げ特例の適用該当基準を満たし証拠書類を添付
  • 商工会・商工会議所の事業支援計画書取得（様式4）を受けること
