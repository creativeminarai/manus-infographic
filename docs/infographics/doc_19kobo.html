第19回公募要領第5版

資料の要約と重要ポイントのまとめ

概要

第19回公募要領第5版は、小規模事業者持続化補助金（一般型 通常枠）に関する最新の公募要領です。2026年1月28日に公開され、補助金は小規模事業者の販路開拓や生産性向上を支援します。補助上限は原則50万円で特例により増額可能。申請は電子申請限定で2026年3月6日から受付開始、4月30日締切です。申請前の要綱熟読とＧビズＩＤプライムの取得が必須です。

重要ポイント

    ['
  • 小規模事業者や特定非営利活動法人を対象に販路開拓・生産性向上の取組に必要な経費の一部を補助
    • ', '
  • 補助上限は50万円、インボイス特例や賃金引上げ特例により最大200万円まで上乗せ可能
    • ', '
  • 補助率は原則2/3、赤字事業者の賃金引上げ特例は3/4
    • ', '
  • 対象経費は機械装置費、広報費、ウェブサイト関連費、展示会出展費、旅費、新商品開発費、借料、委託費など
    • ', '
  • 申請受付期間：2026年3月6日（金）～4月30日（木）17時（予定）
    • ', '
  • 申請は電子申請システムのみ受付、郵送不可
    • ', '
  • 事業支援計画書（様式4）の発行受付は4月16日まで厳守
    • ', '
  • ＧビズＩＤプライムのアカウント取得が申請必須
    • ', '
  • 第三者からの高額な支援アドバイス料金には注意が必要
    • ']

注意点・締切

    ['
  • 補助金は審査あり、不採択や減額の可能性がある
    • ', '
  • 補助金は後払いのため自己負担が先に必要
    • ', '
  • 虚偽申告や補助金の不正受給は法的罰則や交付取消の対象
    • ', '
  • 第三者支援の事実を正確に申告しない場合は不採択や取消となる
    • ', '
  • ＧビズＩＤアカウントの第三者利用は利用規約違反
    • ', '
  • 申請前に「注意事項」や「補助事業者の義務」など要綱の該当ページを熟読必須
    • ', '
  • 応募締切や事業支援計画書発行締切は厳守、期限過ぎると申請不可
    • ', '
  • 小規模事業者持続化補助金＜創業型＞との重複申請は禁止
    • ']

次のアクション

  • まず、最新の公募要領全体を必ず熟読してください（P.1 注意事項、P.23 補助事業者の義務など重点的に）。
  • ＧビズＩＤプライムのアカウントを事前取得し、利用登録を完了させてください（https://gbiz-id.go.jp/top/）。
  • 事業支援計画書（様式4）の発行依頼は2026年4月16日までに済ませましょう。発行不能になるため期限厳守。
  • 自身の所属地区（商工会地区か商工会議所地区か）を確認し、問い合わせ先を把握してください。
  • 申請は電子申請システムを通してのみ可能です。郵送等の方法は利用不可。
  • 第三者支援やアドバイスを受ける場合は契約内容や料金に注意し、正確に申告してください。
  • 申請後は補助事業者の義務を遵守し、虚偽申請や不正受給厳禁であることを理解の上取り組んでください。
