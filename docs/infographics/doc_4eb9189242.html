重要項目の詳細解説

交付の目的

小規模事業者等の生産性向上と持続的発展を図るため、販路開拓等の補助事業に対する経費を補助。

申請から交付決定まで

申請は様式第1で、必要書類と共に商工会地区事務局へ提出。消費税等仕入控除税額を減額して申請。

補助金交付後の管理

補助事業の経理は明確に区分し、帳簿や証拠書類は5年間保存。

申請の取下げと計画変更

不服時は交付決定通知受領後10日以内に取下げ可能。計画変更は条件付きで承認が必要。

補助金の返還・是正措置

不正や規定違反があれば交付決定の取消し、返還命令があり、延滞金も課される。

財産管理と処分制限

取得財産は善良な管理者の注意をもって管理し、処分制限財産は一定期間承認なく処分不可。

個人情報の取扱い

個人情報は厳重管理し、漏えいの場合は直ちに事務局に報告し指示に従う。

反社会的勢力排除の誓約

申請前に反社会的勢力との関係がないことを誓約し、違反時は不利益を免れない。

注意事項・不備対策

  • 申請時に消費税等の仕入控除税額を明確にし減額申請しないと交付不可。
  • 特例適用は全要件満たさなければ補助金が交付されない。
  • 補助金の不正利用や規定違反は取消し・返還命令の対象。
  • 申請書類や報告書は期限厳守で提出必須。
  • 財産処分は承認申請が必要、無断処分は返還義務・加算金対象。
  • 個人情報の漏えいは速やかに報告し指示に従うこと。
  • 反社会的勢力との関係が確認された場合は交付無効となる。
