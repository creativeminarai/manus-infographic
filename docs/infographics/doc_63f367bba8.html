申請の流れ

申請⇒採択⇒見積書提出⇒交付決定⇒補助事業実施⇒実績報告⇒検査・確定⇒請求・入金の順。

審査結果通知

申請締切後2～3ヶ月で事務局HPで公表及び全申請者に通知。採択結果内容に関する個別問合せ不可。

補助対象者の要件

国内居住者で商工会・商工会議所会員非会員問わず申請可能。個人は申請時点で開業している必要あり。

補助対象経費のポイント

ウェブサイト関連費は上限50万円。汎用性のある機械や単なる買換は対象外。中古品は複数見積必須。自動車本体は対象外。

提出書類と様式

様式4の事業支援計画書は発行必須（会員・非会員対応）。様式10は事業承継加点希望時のみ必須。経費明細は具体的に。

特例制度

賃金引上げ特例や小規模事業者卒業加点利用時は、追加の証憑書類が必要。特例要件未達成時は補助金交付されない。

注意事項

発注は交付決定日以降のみ有効。事業期間の延長不可。複数事業の同時申請不可。
