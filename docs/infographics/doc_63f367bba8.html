補助金交付までの流れ

申請⇒審査⇒交付決定⇒事業実施⇒実績報告⇒検査⇒請求⇒入金の順。

審査と結果通知

申請締切後、事務局と有識者による審査。結果は事務局HPにて公表し全申請者に通知。

申請書類と提出方法

電子申請限定。必要書類は事業形態により異なり、開業届や売上台帳なども必要。

補助対象者の要件

国内居住・商工会または商工会議所地区に属する小規模事業者。新規開業予定者は不可。

補助対象経費のポイント

経費区分を厳守し、「一式」等不明瞭表記は禁止。中古品は2社以上の見積提出必須。

特例と加点制度

賃金引上げ特例や小規模事業者卒業加点には要件があり、終了時に条件未達は補助金不交付も。

注意事項・不備対策

  • 申請は電子申請のみで郵送不可。
  • 交付決定日前の発注は補助対象外。

  • 中古品購入は複数見積必須。

  • 要件を満たさない特例申請は不採択。
