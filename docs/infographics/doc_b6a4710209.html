実績報告の作成方法

マイページへログインし、申請内容を確認後、実績報告書入力画面に遷移します。各必要項目（事業概要、期間、経費明細等）を順に入力し、添付資料をアップロードします。

賃金引上げ特例に関する報告

賃金構成（歩合給制、時給制、月給制、年俸制、日給制）に応じた計算フォームを用いて報告。最低賃金労働者情報も記録し証拠資料を添付します。

取得財産等管理明細表

取得資産の名称・規格・数量・単価を入力し、取得年月日と保管場所を正確に登録します。編集・削除も可能。

精算払請求書の作成

実績報告提出後、精算払請求画面より振込先情報の確認・登録を行い、請求手続きを進めます。

注意事項・不備対策

  • 対象補助金は＜一般型 通常枠＞のみ。他補助金には不適用。
  • 紙申請の場合、以降の申請も紙提出となり電子申請不可。
  • 動作環境は最新ブラウザ版に限定。Microsoft EdgeのIEモードは不可。
  • 添付可能ファイル形式を厳守（PDF, ZIP, DOC/DOCX, XLS/XLSX, PNG, BMP, JPG/JPEG, HEIC, GIF）。
  • 支援依頼先（商工会・商工会議所）により手順が異なる場合あり。
  • 各日付入力では正確な実績期間を入力必須。
  • 申請内容は最終確認のうえ提出し、誤りがあれば修正を。
