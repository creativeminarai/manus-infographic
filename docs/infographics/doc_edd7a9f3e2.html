重要項目の詳細解説

補助金とは

国の政策に沿った小規模事業者支援のための公的資金。ルール遵守が支給条件。

補助金支出の原則

  • 補助対象事業を明確に経理区分。
  • 交付決定日以降の発注・支払い。
  • 原則銀行振込支払い。
  • 100万円超は複数見積もり必須（中古品は例外無）。
  • 経費書類の宛名は補助事業者名で統一。
  • 補助金支払いは事業終了後に精算請求書提出をもって完了。

補助事業の流れ

採択通知→見積提出→交付決定→事業実施→実績報告→補助金請求→補助金交付

実績報告・効果報告

  • 事業終了後30日以内または期限までに実績報告書提出必須。
  • 事業効果報告は終了翌月から1年間の観察。
  • 特例・加点対象は賃金・雇用の証拠書類も必須。

証拠書類の必要性

見積書、発注契約書、納品検収書、請求書、支払証明、その他成果物の記録を完備。

特例・加点の注意

  • インボイス特例、賃金引上げ特例の要件１つでも未達成なら補助金不交付。
  • 賃金加点は最低賃金＋30～50円達成が必須。
  • 書類不備は交付取消しの可能性。

契約相手の留意点

100万円超の契約は経済産業省の指名停止措置対象事業者を排除。承認例外あり。

注意事項・不備対策

  • 交付決定日前の発注・支出は補助対象外。
  • 支払は原則銀行振込。小切手・手形・相殺は不可。
  • １取引で10万円超の現金支払いは補助対象外。
  • 不正受給は取消・返還命令、罰則対象。
  • 補助事業者名以外の宛名の書類は不可。
  • 実績報告・効果報告未提出で補助金不交付・次期申請不可。
  • 特例・加点の不適合は全額不支給。
  • 支払遅延や分割払いで期間外決済は対象外。
  • 中古品は必ず複数見積もり取得。選定理由書は不可。

次のアクション

  • 補助対象経費は決定日以降発注・支払分のみ計上。

  • 補助金事務局からの照会に速やかに対応。
