重要項目の詳細解説

補助金とは

国の政策に沿った小規模企業支援資金であり、厳正なルールに基づき交付されます。不正受給は刑罰対象。

補助金支出の原則

  1. 明確な経理区分が必要
  2. 交付決定日以降の経費のみ対象
  3. 原則銀行振込で支払うこと
  4. 100万円超の取引は2者以上の見積もり必須
  5. 書類の宛名は補助事業者名で統一
  6. 補助金は事業完了後に支払われる
  7. 契約相手は経産省の指名停止措置等を確認

実績報告

事業終了後30日以内または最終提出期限までに実績報告書提出が必須。未提出は交付取消となる。

事業効果報告

終了翌月から1年間、効果・賃金引上げ等の状況報告が必須。未提出者は次回申請不可。

証拠書類

段階ごとに見積・発注・納品・請求・支払・成果物を揃え、第三者が確認可能な状態で保存・提出すること。

特例・加点制度

賃上げ特例、インボイス特例、小規模事業者卒業加点などの要件を厳守しないと補助金は交付されません。

注意事項・不備対策

  • 補助金交付決定前の発注・契約は補助対象外
  • 経費は必ず銀行振込で支払う（例外あり）
  • 100万円超の発注は複数見積もり必須、中古品購入はすべて複数見積もり必須
  • 書類の宛名は補助事業者名で統一し証拠書類の漏れに注意
  • 不正受給は刑罰・返還命令の対象
  • 実績報告・事業効果報告は期限厳守
  • 特例・加点要件違反は全補助金不交付の可能性あり
  • 契約先に指名停止措置事業者は使えない（例外あり）
