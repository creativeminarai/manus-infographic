重要項目の詳細解説

補助金とは

国の政策に合致した小規模企業支援のための資金。補助金交付はルール遵守と実績報告が必要。

補助金支出の原則

交付決定日以降の支払いが対象。支払いは原則銀行振込。10万円超の取引は複数見積もりが必要。

補助事業の流れ

採択→見積提出→交付決定→事業実施→実績報告→請求・交付→効果報告の段階を踏む。

実績報告と書類保存

事業終了後30日以内に実績報告。関連書類は最長5年間保管し、監査に備える。

特例・加点の注意事項

賃金引上げやインボイス特例は要件厳守。要件不達成の場合、交付は取り消しになる。

注意事項・不備対策

  • 交付決定前の発注・契約・支払いは全て補助対象外
  • 不正受給や虚偽申請は交付取消・返還命令・罰則の対象
  • 支払い証拠は第三者が内容理解可能な書類で提出
  • 立替払は補助事業者による精算が確認できなければ認められない
  • 現金払いは基本不可、例外は旅費や10万円以下の一部案件のみ
  • 補助事業完了前の概算払いは認められない
  • 契約先は経済産業省の指名停止事業者であってはならない
