重要項目の詳細解説

補助金の意義

国の政策に沿った事業に税金から支給される。無条件の支給ではないため、申請内容と実績の整合が必要。

補助金支出の原則

  1. 補助対象事業の経理は明確に区分
  2. 支払いは交付決定日以降の事業で完了が条件
  3. 銀行振込原則、例外あり
  4. 税込100万円超は2者以上見積もり必須
  5. 書類は補助事業者名義で揃える
  6. 補助金は事業終了後に支払
  7. 契約相手方に制限あり。

実績報告と効果報告の義務

事業終了後30日以内に実績報告。効果報告は終了翌月から1年間必須。未提出の場合は次回申請不可。

事業実施の流れ

採択→見積提出→交付決定→事業実施→実績報告→補助金確定→請求・交付→効果報告。

注意すべき特例等

賃金引上げ特例、インボイス特例、小規模事業者卒業加点は、本補助金の交付に影響。要件不遵守で不支給となる。

  • 不正受給は法的処分の対象。

  • 支払いは基本銀行振込のみ、例外に注意。
